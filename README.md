# 🥗 MenuMestre: Gerador de Cardápios Inteligente

O **MenuMestre** é uma aplicação web que ajuda usuários a organizarem suas refeições semanais com base em orçamento e preferências dietéticas. O projeto utiliza lógica de sorteio sem repetição para garantir uma semana variada e equilibrada.

## ✨ Funcionalidades
- **Filtro por Orçamento:** Opções Econômicas, Padrão e Premium.
- **Filtro por Dieta:** Vegetariana, Fitness/Low Carb ou Sem restrições.
- **Variedade Garantida:** O algoritmo impede a repetição do mesmo prato na mesma semana.
- **Design Responsivo:** Funciona perfeitamente em mobile e desktop.

## 🛠️ Tecnologias
- HTML5
- CSS3 (Flexbox/Grid)
- JavaScript (Vanilla JS)
